# Export Command Design

## Overview

Add `bean-sieve export` command to export parsed transactions to standard CSV/XLSX format. Pure parsing and export, no reconciliation logic involved.

## Use Cases

- Integration with other tools (Excel, other accounting software)
- Data backup and archiving

## CLI Interface

```bash
bean-sieve export <files...> [-p provider] [-o output] [-f csv|xlsx]
```

### Parameters

| Parameter | Description |
|-----------|-------------|
| `files` | Statement files (supports multiple, supports glob) |
| `-p/--provider` | Provider ID (optional, auto-detect by default) |
| `-o/--output` | Output path (optional, auto-generated by default) |
| `-f/--format` | Output format: `csv` (default) or `xlsx` |

### Output File Naming

- If `-o` not specified, generate from input filename: `xxx_export.csv` or `xxx_export.xlsx`
- Multiple input files are exported separately (each generates its own output file)

## CSV Format

### Encoding

UTF-8 without BOM

### Column Order (Fixed)

**Core fields:**

| Column | Description |
|--------|-------------|
| `date` | Transaction date (YYYY-MM-DD) |
| `time` | Transaction time (HH:MM:SS), empty if not available |
| `amount` | Amount, positive = expense, negative = income |
| `currency` | Currency code (CNY/USD etc.) |
| `description` | Transaction description |
| `payee` | Payee / counterparty |
| `card_last4` | Last 4 digits of card number |
| `order_id` | Order ID |

**Metadata fields (appended at end):**

| Column | Source Provider |
|--------|-----------------|
| `category` | Alipay |
| `peer_account` | Alipay |
| `method` | Alipay / WeChat |
| `status` | Alipay / WeChat |
| `merchant_id` | Alipay |
| `tx_type` | WeChat / CGB / PAB |
| `summary` | PAB |
| `posting_date` | CMB / CMBC |

### Excluded Fields

- Processing metadata: `provider`, `source_file`, `source_line`, `post_date`, `account`, `contra_account`, `confidence`, `match_source`, `tags`, `flag`, `statement_period`
- Original date strings: `original_date`, `original_trans_date`, `original_post_date`
- Other: `section`
